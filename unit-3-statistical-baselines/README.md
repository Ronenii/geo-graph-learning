# Unit 3 — Statistical Baselines

**Capability after this unit:** Define and evaluate a univariate
forecasting baseline (SARIMA on bus segment speed). Identify the
breakeven horizon at which the baseline degrades to a historical-
average floor. Articulate the spatial gap that motivates a network
model.
