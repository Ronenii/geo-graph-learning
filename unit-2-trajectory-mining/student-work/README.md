# Unit 2 — student-work/

**This directory is yours.** Put your in-class supervised-practice work,
your homework, and any experimental notebooks here. The upstream public
repo only ships this README and a `.gitkeep` — every other file you add
stays in your fork and survives `git fetch upstream`.

When you push your work to your fork (for async help, or just as a
backup), keep your decision log alongside your code. The instructor can
clone your fork to see your code + your reasoning together.

Expected per-stage files (none of these is required to exist; create
them as you work):

- `practice/` — in-class supervised-practice work for this unit.
- `homework/` — take-home homework deliverables.
- `decision-log.md` — filled per direct → interpret → extend cycle.
- `writeup.md` — your 300–500-word interpretation of results.

The instructor's `promoter` agent **never touches files in this
directory** after the initial scaffold.
