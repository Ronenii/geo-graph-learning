# Unit 2 — Trajectory Mining

(forward inference, HMM map matching) OR infer a network back from raw
GPS alone (inverse, KDE → skeleton → graph). Choose the noise model and
inference direction by question. Articulate when classical methods
suffice and where learned alternatives are moving the field.
